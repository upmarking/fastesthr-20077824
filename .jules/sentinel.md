## 2024-05-24 - XSS in Offer Letter PDF Generation Templates
**Vulnerability:** XSS vulnerability when generating HTML content and PDFs for Offer Letters. \`replaceVariables\` and \`substituteVariables\` accepted unsanitized user inputs (e.g. candidate names) and directly interpolated them into a template string containing structural \`<script>\` elements using \`String.prototype.replace()\`, which then executes scripts when evaluating the DOM tree for HTML2Canvas.
**Learning:** Due to the PDF generator (\`src/lib/pdf-generator.ts\`) intentionally executing \`<script>\` tags from templates for dynamic calculation, stripping the entire HTML tree of unapproved tags using \`dompurify\` strips vital components needed for the template calculations.
**Prevention:** Rather than using a full HTML sanitizer over the entire finalized HTML output string, use an \`escapeHtml\` helper to properly escape only the user-provided variables (to \`&amp;\`, \`&lt;\`, etc) inside the \`substituteVariables\`/\`replaceVariables\` loops *before* they're interpolated.
